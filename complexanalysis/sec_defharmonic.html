    Proposition 6.1.4.

    Suppose \(f=u+iv\) is holomorphic in the region \(G\text{.}\) Then \(u\) and \(v\) are harmonic inÂ \(G\text{.}\)

    Proof.

    First, by CorollaryÂ 5.1.6, \(u\) and \(v\) have continuous second partials. By TheoremÂ 2.3.1, \(u\) and \(v\) satisfy the Cauchyâ€“Riemann equationsÂ (2.3)

    Theorem 6.1.7.

    Suppose \(u\) is harmonic on a simply-connected region \(G\text{.}\) Then there exists a harmonic function \(v\) in \(G\) such that \(f=u+iv\) is holomorphic in \(G\text{.}\)

    ðŸ”—
    The function \(v\) is called a harmonic conjugate ofÂ \(u\text{.}\)
    ðŸ”—

    Proof.

    We will explicitly construct a holomorphic function \(f\) (and thus \(v = \Im f\)). First, let
    \begin{equation*} g \ := \ u_x - i \, u_y \,\text{.} \end{equation*}
    ðŸ”—
    The plan is to prove that \(g\) is holomorphic, and then to construct an antiderivative of \(g\text{,}\) which will be almost the function \(f\) that weâ€™re after. To prove that \(g\) is holomorphic, we use TheoremÂ 2.3.1: first because \(u\) is harmonic, \(\Re g = u_x\) and \(\Im g = - u_y\) have continuous partials. Moreover, again because \(u\) is harmonic, \(\Re g\) and \(\Im g\) satisfy the Cauchyâ€“Riemann equationsÂ (2.3):
    \begin{equation*} \left( \Re g \right)_x \ = \ u_{xx} \ = \ - u_{yy} \ = \ \left( \Im g \right)_y \end{equation*}
    and
    \begin{equation*} \left( \Re g \right)_y \ = \ u_{xy} \ = \ u_{yx} \ = \ - \left( \Im g \right)_x \, \text{.} \end{equation*}
    ðŸ”—
    TheoremÂ 2.3.1 implies that \(g\) is holomorphic in \(G\text{,}\) and so we can use CorollaryÂ 5.2.4 to obtain an antiderivative \(h\) of \(g\) on \(G\) (here is where we use the fact that \(G\) is simply connected). Now we decompose \(h\) into its real and imaginary parts as \(h=a+ib\text{.}\) Then, again using TheoremÂ 2.3.1,
    \begin{equation*} g \ = \ h' \ = \ a_x + i \, b_x \ = \ a_x - i \, a_y \,\text{.} \end{equation*}
    (The second equation follows from the Cauchyâ€“Riemann equations (2.3).) But the real part of \(g\) is \(u_x\text{,}\) so we obtain \(u_x = a_x\) and thus \(u(x,y) = a(x,y) + c(y)\) for some function \(c\) that depends only on \(y\text{.}\) On the other hand, comparing the imaginary parts of \(g\) and \(h'\) yields \(-u_y = -a_y\) and so \(u(x,y) = a(x,y) + c(x)\) where \(c\) depends only on \(x\text{.}\) Hence \(c\) has to be constant, and \(u(x,y)=a(x,y)+c\text{.}\) But then
    \begin{equation*} f(z) \ := \ h(z) + c \end{equation*}
    is a function holomorphic in \(G\) whose real part is \(u\text{,}\) as promised.

    \begin{equation*} f' \ = \ u_x + i \, v_x \ = \ u_x - i \, u_y \,\text{.} \end{equation*}

    Corollary 6.1.10.

    A harmonic function is infinitely differentiable.
    ðŸ”—
    ðŸ”—

    Proof.

    Suppose \(u\) is harmonic in \(G\) and \(z_0 \in G\text{.}\) We will show that \(u^{ (n) } (z_0)\) exists for all positive integers \(n\text{.}\) Let \(r>0\) such that the disk \(D[z_0,r]\) is contained in \(G\text{.}\) Since \(D[z_0,r]\) is simply connected, TheoremÂ 6.1.7 asserts the existence of a holomorphic function \(f\) in \(D[z_0,r]\) such that \(u = \Re f\) on \(D[z_0,r]\text{.}\) By CorollaryÂ 5.1.6, \(f\) is infinitely differentiable on \(D[z_0,r]\text{,}\) and hence so is its real partÂ \(u\text{.}\)
