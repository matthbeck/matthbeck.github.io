Corollary 8.2.5.

Suppose \(f\) is holomorphic in a bounded region \(G\) and continuous on its closure. Then
\begin{equation*} \sup_{ z \in G } |f(z)| \ = \ \max_{ z \in \partial G } |f(z)| \,\text{.} \end{equation*}

Proof.

This is trivial if \(f\) is constant, so we assume \(f\) is non-constant. By the Extreme Value TheoremÂ A.0.1 there is a point \(z_0\in\overline G\) so that \(\max_{z\in\overline G}|f(z)| = |f(z_0)|\text{.}\) Clearly \(\sup_{ z \in G } |f(z)| \le \max_{ z \in\overline G } |f(z)|\text{,}\) and this is easily seen to be an equality using continuity at \(z_0\text{,}\) since there are points of \(G\) arbitrarily close to \(z_0\text{.}\) Finally, TheoremÂ 8.2.4 implies \(z_0\not\in G\text{,}\) so \(z_0\) must be in \(\partial G\text{.}\)
