Proposition 2.2.8.

Suppose \(G, H \subseteq \C\) are open sets, \(f: G \to H\) is a bijection, \(g: H \to G\) is the inverse function of \(f\text{,}\) and \(w_0 \in H\text{.}\) If \(f\) is differentiable at \(g(w_0)\) with \(f'(g(w_0)) \ne 0\) and \(g\) is continuous at \(w_0\text{,}\) then \(g\) is differentiable at \(w_0\) with
\begin{equation*} g'(w_0) \ = \ \frac 1 { f' \left( g(w_0) \right) } \,\text{.} \end{equation*}

Proof.

Since \(f(g(w)) = w\) for all \(w \in H\text{,}\)
\begin{align*} g'(w_0) \amp \ = \ \lim_{w\to w_0}\frac{ g(w) - g(w_0) }{ w - w_0 }\\ \amp \ = \ \lim_{w\to w_0}\frac{ g(w) - g(w_0) }{ f(g(w)) - f(g(w_0)) }\\ \amp \ = \ \lim_{w\to w_0}\frac 1 { \dfrac{ f(g(w)) - f(g(w_0)) }{g(w) - g(w_0) } } \,\text{.} \end{align*}

This is continuous at \(z_0\) and \(\lim_{w\to w_0}g(w) = z_0\) by continuity of \(g\text{,}\) so we can apply PropositionÂ 2.1.11:
