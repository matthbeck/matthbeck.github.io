Example 7.4.8.

Consider the power series \(\ds \sum_{ k \ge 0 } \frac{ 1 }{ k! } \, z^k \text{.}\) Since
\begin{equation*} \lim_{ k \to \infty } \left| \frac{ c_{ k+1 } }{ c_{ k } } \right| \ = \ \lim_{ k \to \infty } \frac{ k! }{ (k+1)! } \ = \ \lim_{ k \to \infty } \frac{ 1 }{ k+1 } \ = \ 0 \, \text{,} \end{equation*}
the ratio test (ExerciseÂ 7.5.32) implies that the radius of convergence of \(\sum_{ k \ge 0 } \frac{ 1 }{ k! } \, z^k\) is \(\infty\text{,}\) and so the power series converges absolutely in \(\C\text{.}\)
