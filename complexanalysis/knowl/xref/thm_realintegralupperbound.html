Theorem A.0.5.

If \(f, g : [a,b] \to \R\) are continuous functions then
\begin{align*} \left| \int_a^b f(x) \, g(x) \, \diff{x} \right| \ \amp\le \ \int_a^b \left| f(x) \, g(x) \right| \diff{x}\\ \amp\le \ \left( \max_{ a \le x \le b } |f(x)| \right) \int_a^b \left| g(x) \right| \diff{x} \,\text{.} \end{align*}
