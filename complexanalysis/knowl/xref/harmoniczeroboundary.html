Corollary 6.2.5.

Suppose \(u\) is harmonic in the bounded region \(G\) and continuous on its closure. If \(u\) is zero on \(\partial G\) then \(u\) is zero in \(G\text{.}\)

Proof.

By (6.1),
\begin{equation*} u(z) \ \leq \ \sup_{z \in G} u(z) \ = \ \max_{z \in \partial G} u(z) \ = \ 0 \end{equation*}
and
\begin{equation*} u(z) \ \geq \ \inf_{z \in G} u(z) \ = \ \min_{z \in \partial G} u(z) \ = \ 0 \, \text{,} \end{equation*}
so \(u\) must be zero in \(G\text{.}\)
