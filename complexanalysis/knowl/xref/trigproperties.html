Proposition 3.4.5.

For all \(z, z_1, z_2 \in \C\text{,}\)
\begin{align*} \sin (-z) \amp = - \sin z\\ \cos (-z) \amp = \cos z\\ \sin (z + 2 \pi) \amp = \sin z\\ \cos (z + 2 \pi) \amp = \cos z\\ \tan (z + \pi) \amp = \tan z\\ \cot (z + \pi) \amp = \cot z\\ \sin (z + \tfrac \pi 2) \amp = \cos z\\ \cos (z + \tfrac \pi 2) \amp = -\sin z\\ \sin \left( z_1 + z_2 \right) \amp = \sin z_1 \cos z_2 + \cos z_1 \sin z_2 \\ \cos \left( z_1 + z_2 \right) \amp = \cos z_1 \cos z_2 - \sin z_1 \sin z_2\\ \cos^2 z + \sin^2 z \amp = 1\\ \cos^2 z-\sin^2 z \amp = \cos(2z)\\ \frac{ d }{ \diff{z} } \sin z \amp = \cos z\\ \frac{ d }{ \diff{z} } \cos z \amp = - \sin z \,\text{.} \end{align*}
